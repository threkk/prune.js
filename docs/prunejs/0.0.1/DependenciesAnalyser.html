DependenciesAnalyser

DependenciesAnalyser

Analyses the dependecies of the project looking for issues regarding the usage of them. It targets the depences stored in the sections dependencies and devDependencies of the package.json. It also scans the node_modules folder looking for packages installed that are not in the package.json.
